#############################################################################   ENGLISH   ###############################################################################################

Hello user, important stuff before booting AudioHarbor:

First of all, install the ACETONE font: "https://www.dafont.com/acetone.font". Just download the font, open it and select "Install". Otherwise the interface will look ugly.
Youtube functions will work from the start but if you want to use the Spotify functions follow these instructions:

1 - Create an Spotify API account here: https://developer.spotify.com/documentation/web-api
2 - Once you have created an account, create an App. Type random stuff to cover the app description
3 - You will receive credentials for your app: Client ID and Client Secret
4 - Open the config.json with a file text editor and type the Client ID and Client Secret in the clientId and clientSecret fields
5 - Now you can look for Spotify playlists of any username!

Spotify custom playlists (like "My liked songs") won't appear in the playlist sleection. Only created playlists by the user will show up.
If you want to download your "My liked songs" you can follow this tutorial: 
"https://www.wikihow.com/Share-Liked-Songs-on-Spotify#:~:text=To%20copy%20the%20link%20to,choose%20Copy%20link%20to%20playlist"

That's all, enjoy!

Hola usuario, cosas importantes antes de arrancar AudioHarbor:

En primer lugar, instale la fuente ACETONE: "https://www.dafont.com/acetone.font". Simplemente descarga la fuente, ábrela y selecciona "Instalar". Si no lo haces la interfaz se verá fea.

2 - Una vez creada la cuenta, crea una App. Escribe cosas al azar para cubrir la descripción de la app
3 - Recibirás credenciales para tu app: Client ID y Client Secret
4 - Abre el config.json con un editor de texto y escribe el Client ID y el Client Secret en los campos clientId y clientSecret
5 - ¡Ya puedes buscar listas de reproducción de Spotify de cualquier nombre de usuario!

Las listas de reproducción personalizadas de Spotify (como "Mis canciones favoritas") no aparecerán en la selección de listas de reproducción. Sólo aparecerán las listas de reproducción 
creadas por el usuario. Si quieres descargar tus "Mis canciones favoritas" puedes seguir este tutorial: 
"https://www.wikihow.com/Share-Liked-Songs-on-Spotify#:~:text=To%20copy%20the%20link%20to,choose%20Copy%20link%20to%20playlist"
